# SocialConnect

## 📌 Sobre o projeto
O SocialConnect é um site desenvolvido com o objetivo de conectar pessoas e facilitar a comunicação entre usuários, oferecendo uma interface simples e moderna.

## 🚀 Tecnologias utilizadas
